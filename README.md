# FitTrack - Antrenman Takip Uygulaması
Bu proje, kullanıcıların spor aktivitelerini kaydetmelerini sağlayan modern bir web uygulamasıdır.

## Özellikler
* LocalStorage ile kullanıcı kaydı ve girişi.
* Kullanıcıya özel antrenman listeleme.
* Silme özelliği ve modern Glassmorphism tasarımı.

## Kullanılan Teknolojiler
* HTML5, CSS3 (Modern UI/UX)
* JavaScript (ES6+)
* LocalStorage & SessionStorage
